My Reflections

Web Technology Course

Overall Experience

The Web Technology course has been a valuable learning experience for me. Through this course, I learned how websites and web applications are designed and developed from both the frontend and backend perspectives.

Learning Outcomes

I gained hands-on experience with HTML and CSS, which helped me create well-structured and visually appealing web pages. Learning JavaScript allowed me to add interactivity and make websites more dynamic. I explored React for component-based development and the MERN stack for full-stack applications. I also learned MySQL for database management and writing efficient queries.

Challenges Faced

One of the key challenges I faced was understanding DOM manipulation and handling asynchronous operations in JavaScript. Integrating the frontend and backend in the MERN stack project was also complex, as it required coordinating multiple technologies simultaneously. Responsive design across different screen sizes using media queries was another area that required significant practice and patience.

Skills Gained

Through this course I developed strong skills in building responsive web pages, creating RESTful APIs, working with React Hooks, and managing databases using both MongoDB and MySQL. I also improved my ability to debug code, read documentation, and apply problem-solving strategies in real-world scenarios.

Team Collaboration Experience

Working as part of a team on the course project gave me valuable experience in collaborative development. I learned how to divide tasks effectively, communicate progress, resolve conflicts in code, and integrate individual contributions into a unified application. This experience closely mirrors real-world software development workflows and greatly improved my teamwork skills.
