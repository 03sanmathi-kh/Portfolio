📘 Assignment 1: Personal Blog Website

For this assignment, I designed and developed a responsive Personal Blog Website using HTML, CSS, and Bootstrap. The website features a modern grid-based layout that organizes blog posts in a clean and visually appealing manner. I implemented hover effects, smooth animations, and interactive elements to enhance user experience. A major focus was responsiveness, ensuring the website adapts seamlessly across desktops, tablets, and mobile devices. During development, I encountered challenges in maintaining consistent layouts on different screen sizes, which I resolved using media queries and flexible design techniques. This project strengthened my understanding of web page structure, styling, responsive layouts, and front-end design principles while improving my confidence in building user-friendly websites.
