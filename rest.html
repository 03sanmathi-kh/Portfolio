📌 Project Overview

Developed a Notes REST API using Node.js and Express to perform Create, Read, Update, and Delete (CRUD) operations on notes. Implemented RESTful routes, middleware-based validation, and JSON data handling to ensure efficient communication between the frontend and backend. Integrated a responsive user interface that enables users to add, view, and delete notes dynamically. This project provided hands-on experience with backend architecture, Express routing, middleware implementation, API testing, and frontend-backend integration while following RESTful design principles.
