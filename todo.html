To-Do List Application

This project is a responsive To-Do List Application developed using HTML, CSS, and JavaScript to help users manage daily tasks efficiently. The application allows users to add new tasks, mark tasks as completed, and delete tasks dynamically through an interactive user interface. To ensure data persistence, browser Local Storage was used so that tasks remain available even after refreshing or reopening the page. During development, the primary challenge was handling DOM manipulation and synchronizing task data with local storage. This project strengthened my understanding of event handling, dynamic content updates, browser storage mechanisms, and responsive web design principles, making it a valuable learning experience in front-end web development.
